# Styles

MDx 提供了 2 种首页样式、3 种文章页样式、4 种文章列表样式。在后台「MDx 主题」-「样式」中可以切换这些样式。

!!! info "更多样式"
    随着 MDx 的更新，MDx 可能会增加更多的样式选择。

## 首页样式

首页样式决定了首页的样式。MDx 提供「默认」和「简单」2 种样式。
